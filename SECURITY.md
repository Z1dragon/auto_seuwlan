# Security

`auto-seuwlan` 会读取本地 `.env` 中的校园网账号和密码，并把它们发送到校园网 Portal。请只在你信任的个人设备上使用。

## 不要公开的信息

- `.env`
- 校园网账号和密码
- 完整 Portal URL
- 真实 `UserIP`
- 真实 `wlanacname` / AP 名称
- 带有认证响应细节的日志

## 日志

脚本会尽量避免打印账号和密码，并会对检测输出中的 URL 做脱敏。但在提交 issue 或截图前，仍应自行检查日志。

## 报告安全问题

如果你发现账号泄露、日志脱敏失效或认证参数泄露风险，请不要直接公开贴出敏感信息。请用脱敏后的最小复现描述问题。
