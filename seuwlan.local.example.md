# Local SEU-WLAN Portal link

Copy this file to `seuwlan.local.md` if automatic Portal detection does not work:

```powershell
Copy-Item .\seuwlan.local.example.md .\seuwlan.local.md
```

Then paste your real redirected login URL below. `seuwlan.local.md` is ignored by git.

```text
https://w.seu.edu.cn/a79.htm?UserIP=10.x.x.x&wlanacname=...
```
